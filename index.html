Dirt ↔ Bedrock Swap

Universal Java Edition Resource Pack — Swap Dirt and Bedrock textures

About the Pack

This resource pack flips the world upside down: dirt becomes bedrock, and bedrock becomes dirt. Pure visual chaos, no gameplay changes. Lightweight and safe for all worlds.

```

Features

  • Swap textures for dirt and bedrock
  • Works on most Java versions (choose correct pack_format)
  • No models, sounds, or scripts — pure texture swap
  • Lightweight and easy to install

Installation

  1. Download the ZIP for your Minecraft version.
  2. Move it to your resourcepacks folder.
  3. Open Minecraft → Options → Resource Packs → enable it.
  4. Enter your world — enjoy swapped textures.
```
